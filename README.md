# Intro

A markdown blog system based on [mongodb](https://www.mongodb.com/), [Koa.js](http://koajs.com/) and [Vue.js](https://vuejs.org/)

# Feature

- 部署简单
- markdown友好
- 简洁的UI

# Get Start

```shell
  # 安装npm依赖
  npm install
  # 启动mongodb
  mongod
  # 开启服务端koa的开发，使用supervisor进行热加载
  npm run dev-server
  # 开启客户端vue的开发，基于vue 2.0
  npm run dev-client
```
